# Kielfix - Static Streaming Platform Clone

A high-fidelity front-end web application that replicates the premium user interface and user experience (UI/UX) of modern streaming services like Netflix. Named **Kielfix**, this multi-page static website is built for web development project evaluations, highlighting clean HTML5 structuring, modern CSS styling, and responsive layout architectures.

---

## 📱 Project Pages & Navigation

This project consists of two core connected page views:
1. **`index.html` (Landing & Login Page):** Features the iconic dark interface, custom background poster wall, and a high-fidelity login card built using translucent layer styling.
2. **`trending.html` (Trending Movies Showcase):** A post-login dashboard view displaying curated movie banners, categories, grid layouts, and horizontal media shelves mimicking a real subscription feed.

---

## 🎨 Project Features

* **High-Fidelity UI Replication:** Accurately clones platform aesthetics, including signature branding hues, dark-mode themes, card depth effects, and hover transitions.
* **Fully Responsive Views:** Engineered with flexible layouts (`Flexbox` and `CSS Grid`) and media queries to adapt cleanly to desktops, tablets, and smartphones.
* **Semantic Code Structure:** Built entirely on proper HTML5 element rules (`<header>`, `<main>`, `<section>`, `<nav>`) ensuring strong architectural clarity.
* **Zero Dependencies:** Developed purely with vanilla web technology—no heavy front-end libraries or frameworks needed.
